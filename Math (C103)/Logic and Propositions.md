> [!info] ***Definition*** 
> A **proposition** is a statement that is either _true_ (denoted $T$) or _false_ denoted (denoted $F$).

![[Logical_Operations.png]]

> [!note] Note
> An implication is true exactly when the **if-part** is *false* or the **then-part** is *true*

![[Logical_Equivalence.png]]

A proposition is **tautology** if it is *always true*. For example, $$P \lor \lnot P$$ 
is a tautology. On the other hand, the proposition is a **contradiction** if it is *always false*. For example $$P \land \lnot P$$ 
is a contradiction, because no proposition cannot be both true and false. 

> [!note] Note 
> If $P$ and $Q$ are logically equivalent, then $P \iff Q$ is a tautology. Similarly, if $P \iff Q$ is a tautology, then $P$ and $Q$ are logically equivalent.
